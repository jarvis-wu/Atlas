# Atlas

Atlas is an iOS map & navigation app that helps you find what you want - Powered by Mapbox. Currently features in implementation/planning includes basic map and navigation, finding specific categories of places such as washrooms, realtime multi-user location sharing, and a map-based lost and found.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites
- [x] Xcode 10.
- [x] Swift 4.2.
- [x] iOS 10 or higher.

### Installing

Go to your working directory and clone the repo by running:

```
$ git clone https://github.com/jarvis-wu/Atlas.git
```

If you have not worked with CocoaPods before, it is a dependency manager that is required by this project. You can install it with the following command:

```
$ gem install cocoapods
```

Once you installed CocoaPods, you can run this to install all required dependencies:

```
$ pod install
```

When the installation is finished, open the .xcworkspace file and begin development or testing.
